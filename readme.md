## Concept

Uses `chezmoi` to manage all dotfiles.

## Installation

* Install `chezmoi` using e.g. `brew install chezmoi`
* Run `chezmoi init https://github.com/tdeekens/dotfiles.git`
* Run `chezmoi diff` to view what would be changed
* Run `chezmoi apply`
